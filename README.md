# Sales Analysis Dashboard – Power BI

## Project Overview
This project is a Sales Analysis Dashboard created using Power BI.  
The dashboard helps analyze sales, profit, orders, customer segments, product categories, and regional performance.

## Tools Used
- Power BI
- Power Query
- DAX
- Excel / CSV Dataset

## Key Features
- KPI cards for Total Sales, Total Profit, Total Orders, and Profit Margin
- Monthly sales trend analysis
- Region-wise and category-wise performance analysis
- Top product analysis
- Interactive slicers and filters

## Insights
- Identified top-performing regions and product categories
- Analyzed monthly sales and profit trends
- Compared sales performance across different customer segments
- Created a clean dashboard for business decision-making
